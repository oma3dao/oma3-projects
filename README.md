# OMA3 Projects

OMA3 is a standards body for spatial web infrastructure, web3 interoperability, and AI agent governance. We develop open specifications that enable trustless collaboration across platforms, organizations, and autonomous systems.

This repository is a public index of OMA3's active technical projects. Each document provides enough context to understand the problem space, our approach, and current status.

## Active Projects

| Project | Summary |
|---------|---------|
| [IWPS](iwps.md) | Inter-World Portaling System — seamless teleportation between virtual worlds |
| [MPAS](mpas.md) | Multi-Party Action Standard — cryptographic multi-approval for any action in any system |
| [OMATrust](omatrust.md) | Reputation and attestation framework for decentralized trust |
| [Universal Manifest](universal-manifest.md) | The missing permission layer — portable, signed identity, credentials, and consent between any two entities |

## Get Involved
