# OMA3 Projects

OMA3 is a standards body for spatial web infrastructure, web3 interoperability, and AI agent governance. We develop open specifications that enable trustless collaboration across platforms, organizations, and autonomous systems.

This repository is a public index of OMA3's active technical projects. Each document provides enough context to understand the problem space, our approach, and current status.

## Active Projects

| Project | Summary |
|---------|---------|
| [IWPS](iwps.md) | Inter-World Portaling System — seamless teleportation between virtual worlds |
| [MPAS](mpas.md) | Multi-Party Action Standard — multi-party authorization for any action in any system |
| [OMATrust](omatrust.md) | Verification protocol for the open internet — cryptographic attestations, identity registry, cross-chain trust |
| [Universal Manifest](universal-manifest.md) | The missing permission layer — portable, signed identity, credentials, and consent between any two entities |

## Repository Access

Some OMA3 project repositories are public. Others are private while work is in early development, member review, or active implementation.

Private repositories are member-access only. When a project is ready for broader public review or implementation, OMA3 may publish additional materials, specifications, examples, or reference implementations.

## Why Participate

OMA3 is a member-governed standards body with a token-based participation and contribution model.

Members may receive OMA tokens for participating in OMA3 meetings, working groups, standards development, software development, documentation, testing, ecosystem adoption, and other approved activities. Member dues may also be used to support token allocations under OMA3’s governance processes.

This model is intended to help active contributors participate meaningfully in standards development, rather than relying only on uncompensated work or large-company sponsorship.
